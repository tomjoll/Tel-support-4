Creating Groups in MOLE

On the Groups page Instructors can create, delete, and manage the availability of course groups. Instructors can provide communication and collaboration tools that only Group members can access.

The Instructor has the option of giving the group access to the following features:
Blogs, Collaboration, Discussion Board, Email, File Exchange, Journal, Tasks, Wikis. Optional grading is also available.

The Groups tool in MOLE allows the organisation of students into Groups of any size. Groups can be created one at a time or in sets. Groups can be designated as Self-Enrol; allowing students to add themselves to a Group, or Manual Enrol; allowing the Instructor to assign students to a Group.

You can also create Group Sets - where you can create a number of groups at the same time, and apply students to them in one go. You can choose to add students in the same way as is described below, with the added functionality of being able to create a set of random enrol group sets, allowing students to be placed in one group at random

Creating Manual Enrol Groups

Under the control panel select;

  1. Users and Groups
  2. Groups
  3. Click Create
  4. Select Manual Enrol
  5. Give your group a name
  6. You can then choose which, if any , of the Blackboard collaborative tools you want being made available to that group
  7. Group set options - you can create a smart view for the group, which allows you to identify those students, easily in the grade centre
  8. The final option is to add group members. Click the Add Users button
  9. Place a tick in the box next to the names you want
  10. Click submit and your group will be available.

Creating a Manual Enrol Group Set

Under the control panel

  1. Select users and Groups
  2. Select Manual Enrol under Group Set
  3. Click Create
  4. Select Manual Enrol
  5. Give your group a name
  6. You can then choose which, if any , of the Blackboard collaborative tools you want being made available to that group
  7. Select the number of groups you want
  8. Group set options - you can create a smart view for the group, which allows you to identify those students, easily in the grade centre
  9. Click submit to go onto the next page to enrol students
  • Click Add users to add students to the groups, and then select the students by clicking in the box next to their name,
  • Or
  • Select randomise enrollments to add students at random
  • Or
  • Submit without adding students, to batch upload students into the groups (advisable for large cohort numbers) See guide here
